Welcome to Random-User-API

API ROUTES

Show a Random User - GET REQUEST : http://localhost:5000/api/v1/user/random

Show All User - GET REQUEST : http://localhost:5000/api/v1/user/all

Save a User - POST REQUEST : http://localhost:5000/api/v1/user/save | SYNTAX JSON AS BODY : {"id":"template", "gender": "template","name":"template", "contact": "template", "address": "template", "photoUrl": "template"}

Update a User - PATCH REQUEST : http://localhost:5000/api/v1/user/update | SYNTAX JSON AS BODY : { "id":3, "updatedUserData":{ "address": "berlin" } }

Bulk Update Users - PATCH REQUEST : http://localhost:5000/api/v1/user/bulk-update | SYNTAX JSON AS BODY : [{"id": 3}, {"id": 4}]

Delete a User - DELETE REQUEST : http://localhost:5000/api/v1/user/delete | SYNTAX JSON AS BODY : { "id":4 }
